## 一、Date

**Date** 对象是JavaScript提供的日期和时间的操作接口。它可以表示的时间范围是，1970年1月1日00:00:00前后的各1亿天（单位为毫秒（`ms`））。

**Date** 对象可以作为普通函数直接调用，返回一个代表当前时间的字符串。即使带有参数，**Date** 作为普通函数使用时，返回的还是当前时间。

```javascript
Date()
// Tue Jul 04 2017 10:15:15 GMT+0800 (CST)

Date(2017, 07, 16)
// Tue Jul 04 2017 10:15:15 GMT+0800 (CST)
```
